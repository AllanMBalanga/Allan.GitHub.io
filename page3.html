Snake Game

A classic Snake game built using Python's Turtle library and OOP. The snake can move using arrow keys, and it can grow after consuming food. The game resets upon collision with walls or its own body, it also updates the current score and high scores depending on the number of food taken. This is done by storing and fetching the score in a text file.

Ping Pong Game

A Ping Pong game built using Python's Turtle library and OOP. The movements of the paddles are W, A, S, D (left) and arrow keys (right). The ball bounces back upon contact with a paddle or either the top or bottom walls. The left and right walls does not allow collisions, resulting in a score and resetting ball position if a ball passess through.

Crossing Turtle Game

A Crossing game built using Python's Turtle library and OOP. The movement of the turtle is solely forward, by clicking W. If the turtle collides with a car object, the game ends. If the turtle successfully crosses the screen, it resets its position and increases the level by 1, resulting in a faster car movement adding more challenge to the game.

Turtle Race Betting

A racing game built using Python's Turtle library that asks the user to pick for the color of the turtle (ROYGBIV) that they think would win, Seven turtles having different colors and running at random generated speeds. The game ends when a turtle reaches the corner, displaying a message box to announce whether the user's turtle won or not.
