Production-Ready E-commerce ShoeShop Backend System (PostgreSQL)

Transformed a family business into a comprehensive REST API system, evolving from an academic project to a production-ready backend architecture. Initially developed using SQLAlchemy ORM and later reconstructed with raw SQL via psycopg2, the system manages complex business workflows through a six-table PostgreSQL database handling customer relationships, service requests, inventory management, and order processing. The API features complete CRUD operations, JWT authentication, bcrypt security, and Pydantic validation with comprehensive testing through Postman. This dual-implementation approach provided deep insights into both ORM abstraction and direct SQL control, demonstrating architectural flexibility and advanced database optimization techniques.
